# AI-Assisted Analog Mixed-Signal Internship

This repository is an AI-prompt-driven reproduction of the [`vsdmixedsignalflow`](https://github.com/praharshapm/vsdmixedsignalflow) project.

The goal is to recreate an analog 2:1 AMUX mixed-signal RTL-to-GDS flow for the SKY130 process using OpenLane, while documenting and driving each step through Codex prompts instead of manual commands.

## Scope

- Reproduce the analog 2:1 AMUX mixed-signal design flow.
- Use SKY130 and OpenLane for RTL-to-GDS implementation.
- Capture prompt-driven setup, execution logs, generated results, and screenshots.
- Keep design assets organized under `designs/design_mux`.

## Repository Structure

```text
designs/
  design_mux/
    src/
      lef/
    results/

```
